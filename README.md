# Analisis_comercial_prueba_tecnica_escenario

Proyecto desarrollado como parte de mi portafolio profesional, basado en un caso práctico de análisis comercial en un entorno retail.
El objetivo fue aplicar técnicas de análisis de datos y visualización para explorar el desempeño de ventas, inventario y rentabilidad utilizando Python y Power BI.

Descripción del proyecto:

A partir de un conjunto de datos que simulan la operación de una empresa retail con tiendas físicas y canal online, se realizó un análisis para comprender el comportamiento de las ventas, el estado del inventario y la rentabilidad de los productos.
El proyecto se orienta en responder preguntas tecnicas del análisis comercial, como la identificación de productos con bajo stock, categorías más rentables y tiendas con menor facturación, integrando procesamiento de datos en Python y visualización en Power BI.

Objetivos del análisis:

Analizar las ventas a lo largo del tiempo y compararlas por canal.
Identificar los productos con mayor ingreso y mayor volumen de ventas.
Evaluar el margen promedio por categoría de producto.
Detectar productos con bajo inventario que continúan registrando ventas.
Analizar el desempeño de las tiendas según su ubicación.

Herramientas utilizadas:

Python (pandas, numpy) para la limpieza y análisis de datos.
Power BI para la creación del dashboard interactivo.
Excel / CSV como formato intermedio de datos.

Dashboard

El dashboard permite analizar el desempeño comercial integrando información de ventas, inventario y rentabilidad. 
A través de las visualizaciones se puede observar la evolución de los ingresos por mes, comparar el desempeño entre el canal físico y online, 
e identificar los productos y categorías con mayor aporte al negocio. Además, el análisis de inventario permite detectar productos con bajo stock 
que continúan registrando ventas, apoyado por filtros que facilitan la exploración por ciudad, mes y canal.
